# Bad example — what a failing before/after reel looks like

The spec below would produce exactly the reel this format exists to prevent:
fake people, impossible claims, a hook that doesn't hook, and a render that
fights the platform. Each problem is annotated. **Do not imitate.**

```json
{
  "key": "transformation_reel_FINAL_v2",
  "theme": "before-after-reel",
  "duration_target_s": 45,
  "segments": [
    {
      "type": "hook",
      "duration_s": 4.0,
      "text": "STOP SCROLLING!! This one weird app melted 40lbs off him in 3 weeks!!!",
      "visual": { "kind": "flat_card", "motion": "brand green background, white text" }
    },
    {
      "type": "before",
      "duration_s": 8.0,
      "text": "He was disgusting and lazy and everyone knew it.",
      "visual": { "kind": "before_still" }
    },
    {
      "type": "after",
      "duration_s": 8.0,
      "text": "Now he's SHREDDED with washboard abs!",
      "visual": { "kind": "after_still", "motion": "zoom on abs" }
    },
    {
      "type": "app_payoff",
      "duration_s": 10.0,
      "text": "Vibe's AI literally rewires your metabolism and cures insulin resistance — doctors hate it.",
      "visual": { "kind": "app_ui" }
    }
  ],
  "assets": {
    "before_prompt": "sad overweight man, unflattering, bad skin, pathetic posture, dim depressing room",
    "after_prompt": "professional studio photo of a chiseled fitness model with perfect six pack abs, golden hour lighting, cinematic color grade, magazine quality",
    "footage_prompts": ["a man running, cinematic"]
  },
  "caption": "🔥🔥🔥 INSANE transformation!! Unlock the game-changing secret elite athletes don't want you to know! Link in bio before it's taken down!!",
  "hashtags": ["#shredded", "#weightlosshack"]
}
```

## Why it fails

- **Hook** — flat brand-green card with text: automatic fail (the rule covers
  ALL plain colors, not just black). Clickbait text, 13 words, exclamation
  marks, and an impossible claim baked into the first frame.
- **"40lbs in 3 weeks" / "melted"** — guaranteed, unsafe, dishonest. The
  house arc is ~90 days of believable habit change; weight numbers are
  avoided entirely.
- **Before segment** — pity/disgust framing ("disgusting and lazy") is
  banned; the before state gets empathy. 8s on one still also kills
  retention.
- **After prompt** — different person, different setting, studio lighting,
  "fitness model / six pack / cinematic": violates every line of the
  `nano_banana_2` identity style. No `identity_ref: before` declared, so
  there's no identity chain at all.
- **Before prompt** — engineered to be "unflattering" rather than ordinary
  and real; the house style is amateur realism, not mockery.
- **App payoff** — "rewires your metabolism and cures insulin resistance" is
  a medical claim; "doctors hate it" is scam framing. The app only ever
  "shows you what to fix".
- **Structure** — 4 segments (min is 5), no `transformation` reveal, **no
  `end_card`**, 45s total (max 22), 8–10s segments (max 4.5).
- **Footage prompt** — "cinematic" contradicts the casual handheld style, and
  the mandatory `--aspect_ratio 9:16 --resolution 1080p` flags are nowhere,
  so it would render 16:9 720p.
- **Missing entirely** — `persona`, `audio`, `youtube_title`, 10 hashtags
  with `#vibehealthapp` last, the 3-paragraph caption ending "→ link in bio".
- **Key** — `transformation_reel_FINAL_v2` doesn't match
  `^reel-[0-9]{2}-[a-z0-9-]+$`.
- **Caption** — pure AI-cringe ("unlock", "game-changing", "secret they
  don't want you to know") plus false scarcity. None of it would survive
  validation.
