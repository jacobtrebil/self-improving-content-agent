# Bad example — what a failing health deck looks like

The deck below would pass a lazy glance but fails validation on almost every
rule. Each problem is annotated. **Do not imitate.**

```json
{
  "key": "hydration-deck",
  "theme": "health",
  "eyebrow": "Hydration and Wellness",
  "slides": [
    {
      "type": "cover",
      "title": "Unlock the game-changing power of optimal hydration for your wellness journey"
    },
    {
      "type": "content",
      "kicker": "Hydration tip number one",
      "title": "Drink more water",
      "body": "Drinking water is good for you and you should do it every day because hydration matters."
    },
    {
      "type": "content",
      "kicker": "Science",
      "title": "Water melts fat",
      "body": "Studies show drinking 3 liters a day guarantees you will lose 10 pounds in two weeks."
    },
    {
      "type": "cta",
      "title": "Download now!",
      "body": "Vibe is an app for health.",
      "button": "Get the app",
      "tag": "Stay Hydrated Today!"
    }
  ],
  "caption": "💧💧💧 Elevate your hydration game! Don't forget to like and follow for more amazing content! #water",
  "hashtags": ["#water", "#h2o", "#drink"]
}
```

## Why it fails

- **Key** — `hydration-deck` doesn't match `^[0-9]{2}-[a-z0-9-]+$`; no deck
  number means no `ai-bg/` pairing and no folder ordering.
- **Slide count** — 4 slides (1 cover, 2 content, 1 CTA). The format is 7.
- **Cover** — 11 words of AI-cringe ("unlock", "game-changing", "wellness
  journey"), no `*highlight*`, no `sub`, and **no `bg`** — a flat-card hook is
  an automatic fail.
- **Kickers** — "Hydration tip number one" is 4 words and limp; beats should
  be punchy ("The trap", "Do this", "Step 01").
- **Content slide 1** — says nothing actionable; "water is good for you" is
  filler, no highlight, no number.
- **Content slide 2** — invented statistic and a guaranteed outcome ("melts
  fat", "guarantees you will lose 10 pounds"). Unsupported medical claims are
  a hard fail; outcome language must be hedged.
- **CTA** — "Download now!" has no payoff restatement; body doesn't name a
  real app behavior; button must be exactly "Download Vibe Health →"; tag must
  be lowercase and benefit-led, not Title Case with an exclamation mark.
- **Brand** — "Vibe is an app" — brand in copy is always "Vibe Health".
- **Caption** — one engagement-bait line instead of hook / recap / pitch with
  "→ link in bio"; only 3 hashtags and `#vibehealthapp` is missing.
- **Missing entirely** — `youtube_title`, `bg_prompts`, cover/CTA `bg` files.
