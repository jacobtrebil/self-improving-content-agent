# Claude Code Instructions

Use `AGENTS.md` as the canonical project instructions.

Before starting any task, read `AGENTS.md`.

If working inside a specific format folder, also read that folder's `format.md`, `prompt.md`, `schema.yaml`, `validation.md`, and the `examples/` files (imitate the `good-*.json`, never the `bad-*.md`).

If working inside a campaign, read that campaign's `brief.md`.
